# 何为建模

数据可以通过其生成所可能遵从的统计过程来建模，其它的大部分建模方式可以分两种

1. 对数据进行简洁的近似汇总描述

Google的PageRank

聚类

2. 对数据中抽取出最突出的特征来代替数据并将剩余内容忽略。

频繁项集 （ frequent itemset ): 比如购物篮中同时出现的小规模项集

相识项 ( similar item )： 共同元素比较高的集合对。 比如将购买过的产品推荐给相似的用户组
